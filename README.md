## How to install
```sh
# npm install
$ npm install -g ptjs-cli
```

## Commands

```sh
# Help
$ pt -h

# Add a new git template project
$ pt add

# Create a project from template
$ pt new

# Template list
$ pt list

# Delete Templates
$ pt delete
```
